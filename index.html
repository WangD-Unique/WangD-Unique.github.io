Blog搭建

2019-10-15

版权声明：本文为原创文章，转载请注明作者和出处。

主要介绍基于Windows10的WSL下hexo+Github的方式完成Blog搭建。所以，在一切开始之前，请先注册Github账号，并安装WSL（请参考本Blog的WSL安装一文）。
hexo官方URL：https://hexo.io/ https://github.com/hexojs/hexo
GitHub官方URL：https://github.com

环境安装

Windows软件安装

官方https://gitforwindows.org/下载git，这里主要使用Git Bash。

WSL环境设置

  1. sudo apt-get install nodejs-legacy npm
  2. sudo npm install -g n
  3. sudo n latest
  4. sudo n stable
  5. sudo n lts
  6. sudo npm install npm@latest -g
  7. sudo npm install --save core-js@3
  8. sudo npm install -g hexo
  9. sudo npm install hexo-deployer-git --save
  10. sudo node -v
  11. sudo npm -v

Module方式（推荐）

VS Code Remote - WSL（C语言）

2019-06-01

版权声明：本文为原创文章，转载请注明作者和出处。

VS Code插件安装

以下插件都应安装在WSL中：

  • C/C++，发布者: Microsoft
  • CoolFormat，发布者: WuHuan（如未找到，请参考下文并手动安装）
  • Push, 发布者: N Anderson
  • Text Marker (Highlighter)，发布者: Ryuichi Inagaki

CoolFormat安装：

  1. 打开URL：https://sourceforge.net/projects/coolformat/files/
  2. 至对应的路径下载以下3个文件：vscode-coolformat-0.0.1.vsix、CoolFormatExec1.0(Linux64).zip、CoolFormat3.5.rar
  3. 手动安装vscode-coolformat-0.0.1.vsix
  4. 解压缩CoolFormatExec1.0(Linux64).zip，并拷贝/usr/local/sbin下
  5. 解压缩CoolFormat3.5.rar，这个工具是在Windows下生成CoolFormatConfig.cfconfig文件的
